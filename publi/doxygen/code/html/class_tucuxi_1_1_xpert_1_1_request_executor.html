Detailed Description

This step takes the adjustment trait in the XpertRequestResult and executes it, then makes a new request to extract statistics at steady states and requests to get parameters at "previous types".

void Tucuxi::Xpert::RequestExecutor::gatherAdditionalData ( XpertRequestResult &  _xpertRequestResult) const
protected

This method is called after the first request to Tucuxi core succeed (in perform). It collects the statistics at steady state and the parameters for previous prediction types. To do so, it creates new traits the extend the end date to reach steady state or that change the parameters type.

_xpertRequestResultXpertRequestResult to set the additional data and to get the base adjustment trait.
