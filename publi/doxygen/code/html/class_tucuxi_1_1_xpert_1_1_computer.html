Entry point of the TuberXpert command Line Interface. This method imports the query from a file, loads the translation file, and processes each xpertRequest to finally print the reports of the successfully processed requests.

Parameters
_drugPathPath to the folder containing the drug models.
_inputFileNamePath to the query file.
_outputPathPath ot the output directory. One file is created per successful request.
_languagePathPath to the folder containing the translations files.
Returns
A computingStatus that depends on whether the query could be loaded and how much requestXpert was successfully processed.
