Detailed Description

This class is responsible for checking the plausibility of each dose. It does this by comparing each patient's doses to the doses recommended by the drug model.

The current implementation considers every dose. A future improvement for a specific drug could be to consider only doses that are no older than X years/months/days...

It assumes that the LanguageManager is loaded with a complete translations file.
