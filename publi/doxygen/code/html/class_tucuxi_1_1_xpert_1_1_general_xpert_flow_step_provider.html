Detailed Description

General provider of xpert flow step (generically for the drugs). It provides instances of:

  • 1) CovariateValidatorAndModelSelector
  • 2) DoseValidator
  • 3) SampleValidator
  • 4) TargetValidator
  • 5) AdjustmentTraitCreator
  • 6) RequestExecutor
  • 7) ReportPrinter
