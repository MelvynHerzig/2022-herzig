Detailed Description

This step selects the exporter that matches the desired output format of the XpertRequest. In other words, it selects between the XML, HTML or PDF exporter and send him the the XpertRequestResult to be exported.

This class chooses between:

  • XpertRequestResultXmlExport
  • XpertRequestResultHtmlExport
  • XpertRequestResultPdfExport
