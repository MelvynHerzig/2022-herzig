Detailed Description

This class stores the validation result for a patient covariate. This class is created during the "covariate validator and model selector" phase. It's purpose is to tell what the computing core will receive:

  • The value of the drugmodel. or
  • The patient value.

When the patient value is used, it is possible to set a warning if the validation of the covariate definition is not respected.
